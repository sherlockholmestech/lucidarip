# LucidaRip

LucidaRip is a music ripping tool for Tidal, Qobuz, and Deezer. It is based of the [lucida](https://git.gay/lucida/lucida) library.

## Dependencies
- nodejs
- ffmpeg

## Installation
```bash
git clone https://github.com/sherlockholmestech/lucidarip.git
cd lucidarip
npm install
```

## Usage
```bash
npm run dev [url]
```
